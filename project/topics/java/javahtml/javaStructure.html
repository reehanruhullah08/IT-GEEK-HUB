Structure of a typical Java language



Syntax:

Package statement
important statements
class comments (optional)
class declaration
{

          Instance variable declaration
          Constructor declaration
          Method declaration
}



Example:

Let's have a quick look at Java programming example. A detailed description of the Hello Java example is available here.

    Class first
{
  Public static void main (String args[ ])
   { 
      System.out.println("Hello Java");
   }
}


Output:- Hello Java
  

  Class - It is a key-word used to define a class.
  First - It is a class name (User defined)
  Public - It is an access specifier.This is accessible anywhere (visible to all)
  static - It is an access modifier that bridges the method into memory without object creation.
  void - this is a return type this function will not return anything.
  main - It is a method name. It states execution.
  String - It is a library class (java.lang.string).
  Args -  It is an array name (Identifier).
  ( ) -  A subscript.
  System - It is a library class (system class is present in large package java.lang.system).
  out - A Static data member of system class.
  println - It is a method name.
  Hello Java - String constant (message).
  System.out.println - This is used to print anything.
